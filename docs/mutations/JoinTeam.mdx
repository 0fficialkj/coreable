# JoinTeam

## Declaration

> joinTeam() {}

## Description

> The method used by a logged in user to join an team with an invite code
 
## Pre Conditons

> The user must be logged in
>
> The user must not already be a member of the team
>
> The invite code must be correct

## Parameters

> | Field | Type  | Required? |
> |:-:|:-:|:-:|
> | inviteCode | String | Yes |

## Shell 

```graphql
mutation {
  joinTeam(inviteCode: "") {

  }
}
```
