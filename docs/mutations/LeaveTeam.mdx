# LeaveTeam

## Declaration

> leaveTeam() {}

## Description

> The method used by a logged in user to leave a team they're in

## Pre Conditons

> The user must be logged in
>
> The user must be a member of the team
>
> You must have the team_id

## Parameters

> | Field | Type  | Required? |
> |:-:|:-:|:-:|
> | team_id | String | Yes |

## Shell 

```graphql
mutation {
  leaveTeam(team_id: "") {

  }
}
```
