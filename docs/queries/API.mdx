# API

## Declaration

> API {}

## Description

> An unprotected end point of the API to return the server time and server environment

## Pre Conditions

> None

## Parameters

> None

## Shell

```graphql
query {
  API {

  }
}
``` 

## Example Key

> | Icon | Meaning  |
> |:-:|:-:|
> | # field  | Field always returns null |
> | # field {}  | Subquery always returns null |
> | # =>[Class]  | Subquery returns a list  |
> | # =>{Class}  | Subquery returns an object  |
> | //  | Comment  | 

## Example

```graphql
query {
  API {
    data {
      API { # =>{API}
        env
        time
      }
    }
    errors {
      code
      path
      message
    }
  }
}
```
